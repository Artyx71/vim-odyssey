# ⚔️ Vim Odyssey

*[English version](README_en.md)*

**Vim Odyssey** — это интерактивная веб-игра для изучения и отработки команд редактора Vim. Проект превращает освоение мощнейшего текстового редактора в увлекательное путешествие по мирам с нарастающей сложностью.

## 🚀 Особенности (Features)

- **Настоящий движок Vim:** Полностью кастомный обработчик нажатий клавиш без использования сторонних библиотек. Поддерживает `Normal` и `Insert` режимы.
- **Продвинутые команды (Vim Motions):**
  - Базовая навигация: `h`, `j`, `k`, `l`, `w`, `b`, `e`, `0`, `$`, `^`, `G`
  - Команды с ожиданием (Pending keys): поиск символов `f`, `F`, `t`, `T`
  - Операторы: копирование `yy`, удаление `dd`, изменение `cc`, вставка `p`
  - Текстовые объекты (Text Objects): изменение или удаление внутри кавычек или скобок (например, `ci"`, `di{`, `yi(`).
- **Игровая прогрессия:** Различные миры и уровни (Challenges). Чтобы открыть следующий мир, нужно успешно пройти все испытания в текущем.
- **Поддержка нескольких языков (i18n):** Встроенная система локализации, позволяющая переключать языки на лету.

## 🏗 Архитектура и Технологии

Проект построен с акцентом на производительность, модульность и отказ от лишних зависимостей:

- **Frontend:** React 18 + Vite
- **Управление состоянием:** React Context API + `useReducer` (сохранение прогресса в `localStorage`).
- **Отсутствие тяжелых зависимостей:** Роутинг и стейт-менеджмент реализованы нативно.
- **Изолированный движок:** Логика парсинга команд Vim полностью отделена от UI-компонентов (чистые функции, иммутабельное обновление состояния).

## 🛠 Запуск проекта локально

Убедитесь, что у вас установлен [Node.js](https://nodejs.org/) (рекомендуется версия 18+).

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Artyx71/vim-odyssey.git
   cd vim-odyssey
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```

4. Откройте браузер по адресу, указанному в терминале (обычно `http://localhost:5173`).

## 🎮 Как играть

1. Нажмите "Начать" на главном экране.
2. Выберите доступный мир и уровень.
3. Перед вами появится редактор с текстом и заданием.
4. Выполните поставленную задачу, используя **только** клавиши Vim.
5. Наслаждайтесь процессом и прокачивайте свою мышечную память!
